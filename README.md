# Splurty

Get a quote and submit your own. Built as a part of Berkeley Coding Bootcamp curriculum. 

**Built with:** Ruby on Rails, PostgreSQL, Bootstrap 4, Devise, Simple_form

## Key features
* User authentication
* Retrieve random quote from the database
* Submit your quote
